---
title: 'Custom Ranks'
description: 'Custom Ranks are user defined ranks that replace Hypixel&apos;s default ranks.'
---

<img src="/images/custom-ranks.gif" />

<Note>
    This feature is only available with [Lilith Ultimate](https://lilith.rip/pricing).
</Note>

To get started using your own customized rank follow these steps:

<Steps>
    <Step title={"Support Lilith"}>
        Support Lilith by subscribing to [Lilith Ultimate](https://lilith.rip/pricing), and get access to Custom Ranks.
    </Step>
    <Step title={"Connect to Lilith"}>
        Connect to Lilith with Minecraft `1.8.9`.
    </Step>
    <Step title={"Define your Custom Rank"}>
        To define your Custom Rank's content, use the command `/lrank set <content>`.
    </Step>
    <Step title={"Choose your color"}>
        You can choose which color your Custom Rank will be by using the command `/lrank color <color code>`.

        A color code is Minecraft color code prefixed with `&` instead of `ยง`.
    </Step>
    <Step title={"Toggle your rank"}>
        Toggle your rank using the command `/lrank toggle`.
    </Step>
</Steps>
