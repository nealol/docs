## Description

This command sets the color or the name of your custom rank and toggles your custom rank.

Default Hypixel ranks are not accepted. A Lilith Ultimate subscription is required.

## Usage

`/lrank <toggle|set|color> [arg]`

## Aliases

`/lrank`
